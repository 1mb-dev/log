The last release shipped over two years ago.

The changelog just stops.

Reads as abandoned. A maintainer who moved on.

Clone it today. It still runs. The one thing it does, it still does.

It didn't stop growing because anyone gave up. It stopped because there was nothing left to add. The scope was small on purpose.

Feature-complete is a phrase nobody trusts anymore. Every tool is supposed to grow. More flags. More integrations. A config format. A cloud tier. A logo.

But growth is a decision. So is stopping.

A tool that does one thing and stops there isn't unfinished.

It's finished.

Finished isn't the same as abandoned.
