The old project still runs.

Three years since the last issue. No stars this month. No PRs. The last commit was a dependency bump, and even that felt like overkill.

It just works.

Nobody opens threads about things that don't break. No postmortem. No blog post. No "what we learned."

The logs look the same every week. Graphs flat. Uptime climbing.

The builder moved on years ago. Fixed bigger bugs. Built newer things. This one kept doing what it was built for.

Nobody notices it's there until something breaks. Which it doesn't.

The loudest projects get rewritten every two years. The quiet ones run for a decade.

Most underrated kind of shipping: the thing that never needed a second version.
